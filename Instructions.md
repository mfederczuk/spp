# spp Directive Instructions ##

* `insert`  
  Usage: `#insert <filepath>`

* `include`  

* `exclude`  

* `endexclude`  

## `insert` ##

Synopsis:

```txt
#insert <filepath>
```

## `include` ##

```txt
#include <filepath>
```

## `exclude` ##

Usage: `#exclude [ [next] <n> | [prev] -1 ]`

## `endexclude` ##

```txt
#endexclude
```
